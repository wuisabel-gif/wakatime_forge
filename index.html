WakaForge Cron Explanation

GitHub Actions operates scheduling using UTC. Your selected interval is converted to a simple cron syntax automatically. You will never have to manually write standard cron strings like 0 0 * * * inside your configurations.

Step-by-Step Integration Guide

Go to your WakaTime Account Settings page.

Ensure that Display Coding Activity and Display Languages are visible publicly in settings, and copy your secret API key.

Navigate to your public GitHub profile repository settings.

🔑 WAKATIME_API_KEY: Paste your key from Step 1.
🔑 GH_TOKEN: Create a GitHub PAT with repo scope & paste here.

Open your repository's README.md, paste the generated README Markers anywhere on the page, and commit changes.

Create a file inside your repository named .github/workflows/waka-stats.yml and paste the copy-ready workflow.
